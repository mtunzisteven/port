Week 2: similar to var, but has block scope. Const----** Has block scope. When it is used to declare variable that is a primitive data type, the value assigned to it cannot be changed. In contrast, when used to declare non-primitive data types, the underlying values may be changed. The object can't be reasigned. example: const x = 10; x = 20; // Error! cannot be reasisgned const arrX = [1,2,3,4]; arrX = []; // Error! cannot be reasigned arrX[0] = 12; // [12, 2, 3, 4] If you assign a non-primitive value to a variable, it is done by reference. Any change made to the variable will cascade to the non-primitive value. example: const y = {value:7}; let z = y.value; // z.value = 7, y.value = 7 z.value = 10; // y.value = 10 as well. Changes to the value property of z also led to changes to the value property of y. IndexOf(): Used to find the index of a character or a substring within a string. -1 is returned if the character or string doesn't appear in the string. example: const h = "Hello"; h.indexOf('H'); // 0 h.indexOf('z'); // -1 Includes(): Used to find out if a character is in a certain string. Returns a boolian result Alternates include: startsWith(), endsWith() example: const h = "Hello"; h.Includes('H'); // true h.Includes('z'); // false Template Literals----** Use the back tick in place of quotes: `Hello` insted of "Hello" or 'Hello' Both single and double quotes can be used within them. Create: const setName = new Set(); No literal notation to create it. add an element: setName.add(value); // single item setName.add(value1).add(value2).add(value3); // multiple items const setName =new Set([value1, value2, value3]); // multiple items in one go add complete string words: Use setName.add(value) method, not const setName = new Set([value]). Otherwise the word will be broken into char elements. sets can contain arrays as values, and those value within arrays can be identical. sets do not have indexes. We can use the spread operator to convert a set to an array: const arr = [...setName]; We can also use Array.from to convert a set to an array: const arr = Array.from(setName); We can also use a neat trick to duplicate an array without any repeated elements using a set: const arr2 = [...new Set(arr1)]; // arr2 is the same as arr1, with the exception that no elements are ever repeated. size(): Gives the number of elements in a set example: setName.size(); // integer value for number of elements has(): Can be used to check if a certain value is found within the set.
