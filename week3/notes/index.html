Namespacing

It is possible to encapsulate all related functions in an object in order to avoid name collision:

const myMaths ={
square(z){return z*z;},
add(a,b){return a+b;},
divide(x,y){return x/y;}
};

myMaths.divide(4,2); // 2
myMaths.add(1,2); // 3
myMaths.square(7); // 59

Legacy DOM shortcut methods

Document.images returns all elements

Document.links returns all and elements with href attribute

Document.forms returns all

elements

The returned lists above are array like, but they lack built in functions like splice, etc.
