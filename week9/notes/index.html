THe data attribute

Helps us embed data in a web page using custom attributes that are ignored by the browser.
They're also private to the page, so are not intended to be used by external resources.
Start with data-, contain only lower case letters, hypens, numbers, colons, underscore, or dots. May contain optional string values
Example:

data-user = 'mtunzism'

...div id='user' data-user='mtunzism'> username
const user = document.getElementById('user');
const username = user.dataset.user;

hyphenated names: data-last-name -> dataset.lastName (camel case used to access)
