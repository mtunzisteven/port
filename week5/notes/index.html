Strict Mode
Prevents the use of deprecated features and produces more exceptions and errors. It's the opposite of sloppy mode.
use this on first line of js: 'use strict';
you can also use it per function as follows:

function strictly(){
'use strict';
// function code goes here
}

The recommended way is to put all code into a self invoking funtion. (function() {
'use strict';

// All your code would go inside this function

}());
debugger: this keyword creates a breakpoint where your program pauses while you inspect it in the browser.
Remember to remove it after use.
