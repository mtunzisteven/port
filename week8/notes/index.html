Transforms

The CSS3 transform property lets you lets you translate, rotate, scale, and/or skew any element. We use transofrm functions to manipulate the appearance of an element. The value of the transform is one or more transofrm functions seperated by spaces. The fucntions are applied in the order they appear.

translate()

This fucntion is used to move an element along the x and y axis. It is applied as shown in the following example:

transform: translate(x,y);
transform: translateX(x);
transform: translateY(y);

Transforms don't work on inline elements. Convert inline elements to "inline-block".

scale()

This fucntion is used to scale an element along the x and y axis. When only one value is provided without specifying the axis, the element is scaled while the aspect ratio is maintained. The center of the element is always maintained.It is applied as shown in the following example:

transform: scale(x,y);
transform: translateX(x);
transform: translateY(y);
transform: scale(v);

transform: translate(x,y) scale(x,y);

rotate()

To rotate elements on their point of origin, we use the rotate function. we specify the values in degrees(deg). Positive rotates elements clock-wise, while negative rotates elements anti-clock-wise.

In addition to degrees, values can be provided in grads, radians, or turns

transform: rotate(Vdeg);
transform: rotate(Vdeg) translate(x,y) scale(V);

skew()

This fucntion is used to skew an element along the x and y axis. It is applied as shown in the following example:

transform: skew(x,y);
transform: skew(x.y) rotate(Vdeg) translate(x,y) scale(V);

transform-origin

This fucntion is used to change the origin of an element. It has similar syntax as background-position property. It is applied as shown in the following example:

transform-origin: x% y%;
transform: skew(x.y) rotate(Vdeg) translate(x,y) scale(V);

Transforms don't cause document reflow, therefore, none of the elements neighboring the transformed element will accomodate the transformation. Keep this in mind.

Transitions

Transitions allow the values of CSS properties to change over time. They are used to create animations. All transforms above can be animated. CSS transitions are declared along with the regular styles on an element. Whenever the target properties change, the browser will apply the transition making the change gradual.

Steps to create a simple transition using only CSS:

1. Declare the original state of the element in the default style declaration.
2. Declare the final state of your transitioned element; for example, a :hover state.
3. Include the transition functions in your default style declaration using the

transition properties, including: transition-property, transition-duration, transition-timing-function, and transition-delay.

transition-property

The transition-property property defines the CSS properties of the element that should be transitioned, with all for all properties being the default.

Any property changing from one value to another for which you can find a valid midpoint can be transitioned. For example, in transitioning from a 1px red border to a 15px blue border, we transition the color and width of the border.

It is important to include a pre-state and a post-state. For example, to transition from rectangular corners to rounded corners, set the original state to border-radius: 0;. From the above examples, we can use it in the following way:

transition-property: transform;
transition-property: transform, color;

transition-duration

The transition-duration property sets how long the transition will take: the duration of time it takes to go from the default state to the transitioned state. You can specify this either in seconds (s) or milliseconds (ms). any duration lower-than 200ms makes the website seem slow. Anything faster may be too quick to see. To specify 0.2 seconds (0.2s), or 200 milliseconds (200ms) we use the following:

transition-duration: 0.2s;

transition-timing-function

The transition-timing function properties set the pace of the transition. they have key therms:

ease, linear, ease-in, ease-out, or ease-in-out. The default is ease.

transition-timing-function: ease-out;

transition-delay property

The transition-delay properties is used to add a delay before a transition begins. The default is 0. Use ms or s to specify the delay.

transition-timing-function: ease-out;
