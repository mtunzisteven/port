Forms

We can access the form in JS using the following line: const varNameForForm = document.forms['FormNameAttribute'][index];
Without an index, an array of all forms in the document is returned.
Forms index is based on order of appearance in the html document

We can add an event listener the the form using the forllowing line: varNameForForm.addEventListener('submit', functionName, false);

To access checked input elements from a checkbox list:
let varNameForArrayOfInputs = [...event.target.['inputAttrName']].filter(box => box.checked).map(box => box.value);

This uses the spread operator to turn the node list into an array. This then allows us to use the filter() method that returns an array containing only the check boxes that were checked (this is because their 'checked' property will be truthy). We then chain the map() method to the end, which replaces each checkbox in the array with its 'value' property. This array is then returned and stored in the hero.powers variable.
